# My Blog

Starter landing page + blog built with the latest Next.js (App Router + TypeScript + Tailwind CSS).

## Run locally

1. Install Node.js 20+.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Routes

- `/` landing page
- `/blog` blog index
- `/blog/[slug]` individual blog post
